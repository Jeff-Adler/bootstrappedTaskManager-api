# Bootstrapped TaskManager

Basic TaskManager app using the starter pack found at: https://github.com/Jeff-Adler/Express_TypeScript_TypeORM_PSQL_StarterPack_v3

# Technologies

Node
Express
TypeScript
TypeORM
PostgreSQL

# Routes:

- Login
- Logout
- Create task
- Complete task
- Get all of my (completed/uncompleted) tasks
- Edit task
- Delete task
